# CPTS Note Taker

![HTB Certified Penetration Testing Specialist banner](htb.jpg)

Single-file HTML note taker for CPTS study prep.

## Overview

`CPTS_NoteTaker.html` is a browser-based study companion for organizing CPTS module notes in one place. It provides a dark, compact interface with a left-side module navigator and a main workspace for capturing notes, commands, and exam checklist items.

## Features

- Module list for the CPTS curriculum, including search and quick navigation.
- Separate tabs for module notes, reusable commands, and exam checklist items.
- Local browser storage so your notes persist between sessions.
- Progress tracking with module status indicators and completion controls.
- Export button to download notes for backup or review.

## How to use

1. Open `CPTS_NoteTaker.html` in a browser.
2. Select a module from the sidebar.
3. Add notes, commands, and checklist items in the tabs on the right.
4. Use the export button to save your notes when needed.

## Files

- `CPTS_NoteTaker.html` - main note-taking application.
- `CPTS_Path.md` - supporting path/reference notes.
